Image Search Abstraction Layer

Part 1. Insert the Searchterm in the URL and get the corresponding images. Within the URL parameter "offset" you can paginate through the responses.

Part 2. With a second URL you can display the last 10 search terms.

How to use image search?

Example URL: https://daa-image-search-api.glitch.me/api/search/iyengar yoga horgen?offset=1
